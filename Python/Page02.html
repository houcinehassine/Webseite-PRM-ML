Matplotlib Plotten

matplotlib.pyplot (als plt) erzeugt alle Visualisierungen im Kurs — von einfachen Scatter-Plots bis zu Entscheidungsflächen und Bildrastern.

plt.scatter plt.plot subplots contourf imshow

Worum geht's?

Die wichtigsten Plot-Funktionen und die wiederkehrenden Visualisierungs-Rezepte (Entscheidungsfläche, Elbow-Plot).

2.1 Figur & Achsen

Abbildung anlegen und abschließen.

plt.figure / plt.subplots matplotlib

figure(figsize=(b,h)) öffnet eine neue Abbildung; subplots(zeilen, spalten) erzeugt ein Raster aus Achsen (ax).

import matplotlib.pyplot as plt
plt.figure(figsize=(8, 5))
# ... oder mehrere Achsen:
fig, axes = plt.subplots(1, 2, figsize=(11, 4))
axes[0].plot(x, y)
Im Kurs: Jede Visualisierung startet hier — Größe steuern, mehrere Plots nebeneinander.
plt.show / tight_layout / close matplotlib

show() zeigt die Abbildung; tight_layout() verhindert Überlappungen; close('all') gibt Speicher frei.

plt.tight_layout()
plt.show()
Im Kurs: Standard-Abschluss jeder Plot-Zelle. Im Browser (Pyodide) fängt der Code show() ab und rendert das Bild.

2.2 Plot-Typen

Die Diagrammarten, die im Kurs vorkommen.

plt.scatter(x, y, c=..., cmap=...) matplotlib

Punktwolke. Mit c=labels und cmap färbt man Punkte nach Klasse/Cluster; s=... steuert die Größe.

plt.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm',
            edgecolor='k', s=25)
# Größe ~ Wahrscheinlichkeit:
plt.scatter(X[:,0], X[:,1], s=proba*120)
Im Kurs: Klassen/Cluster visualisieren, GMM-Wahrscheinlichkeit als Punktgröße (Blatt 06).
plt.plot(x, y, fmt) matplotlib

Linien-/Punktdiagramm. Das Format-Kürzel kombiniert Farbe, Marker, Linienstil (z.B. 'o-', 'r--').

plt.plot(ks, inertias, 'o-')          # Elbow-Plot
plt.plot(fpr, tpr, label='ROC')
plt.plot(sizes, train_sc, 'b-o')      # Lernkurve
Im Kurs: Elbow-Plot, ROC-Kurve, Lernkurve, BIC/AIC über k.
plt.hist / imshow / contourf matplotlib

hist = Histogramm, imshow = Bild/Matrix als Heatmap, contourf = gefüllte Höhenlinien (Entscheidungsflächen).

plt.hist(werte, bins=30)
plt.imshow(importances.reshape(8, 8), cmap='hot')
plt.contourf(xx, yy, Z, alpha=0.25, cmap='coolwarm')
Im Kurs: Verteilungen, Feature-Importance-Heatmaps (Blatt 05), Entscheidungsflächen (Blatt 03).

2.3 Beschriftung & Deko

Plots lesbar machen.

Titel & Achsen matplotlib

Beschriftungen für Lesbarkeit. suptitle setzt einen übergeordneten Titel über mehrere Subplots.

plt.title('SVM Entscheidungsfläche')
plt.xlabel('Feature 1'); plt.ylabel('Feature 2')
plt.suptitle('Generierte Ziffern')
Im Kurs: Jeder Plot bekommt aussagekräftige Beschriftungen.
legend / colorbar / grid / axvline matplotlib

legend() zeigt Labels, colorbar() die Farbskala, grid() ein Gitter, axvline(x) eine vertikale Markierung.

plt.legend()
plt.colorbar()
plt.grid(alpha=0.3)
plt.axvline(best_k, color='red', ls='--')   # bestes k markieren
Im Kurs: Bestes k im Elbow-Plot markieren, Farbskala bei Heatmaps.
ax.axis('off') / subplot-Grid matplotlib

Bei Bildrastern blendet axis('off') die Achsen aus — typisch für generierte Bilder.

fig, axes = plt.subplots(3, 4, figsize=(7, 5))
for ax, img in zip(axes.ravel(), bilder):
    ax.imshow(img.reshape(8, 8), cmap='gray')
    ax.axis('off')
Im Kurs: 3×4-Raster generierter Gesichter/Ziffern (Blatt 06, PCA+GMM).

2.4 Wiederkehrende Rezepte

Zwei Muster, die du immer wieder brauchst.

Muster: Entscheidungsfläche Rezept

Das wiederkehrende Rezept, um zu zeigen, wie ein Modell den Raum aufteilt: Gitter bauen → vorhersagen → contourf.

xx, yy = np.meshgrid(np.linspace(x0, x1, 300),
                     np.linspace(y0, y1, 300))
Z = model.predict(np.c_[xx.ravel(), yy.ravel()]).reshape(xx.shape)
plt.contourf(xx, yy, Z, alpha=0.25, cmap='coolwarm')
plt.scatter(X[:,0], X[:,1], c=y, edgecolor='k')
plt.show()
Im Kurs: Standard-Visualisierung für SVM, Entscheidungsbäume, Clustering.
Muster: Elbow-Plot (One-Liner) Rezept

Inertia über k plotten, um die optimale Clusterzahl zu finden — geht sogar in einer Zeile.

plt.plot(range(2, 11),
         [KMeans(k, n_init=10).fit(X).inertia_ for k in range(2, 11)], 'o-')
plt.xlabel('k'); plt.ylabel('Inertia'); plt.show()
Im Kurs: Optimale Clusterzahl bestimmen (Kapitel 4, Blatt 04).
