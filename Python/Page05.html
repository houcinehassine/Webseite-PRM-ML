5.1 Support Vector Machines

Maximaler Margin, linear oder mit Kernel.

SVC sklearn.svm

Support Vector Classifier. Trennt Klassen mit maximalem Margin; mit Kerneln (linear, rbf) auch nichtlinear.

  • C — Strafterm: groß = schmaler Margin (Overfit-Gefahr)
  • kernel — 'linear' oder 'rbf'
  • gamma — RBF-Reichweite: 'scale'/'auto'/Zahl
from sklearn.svm import SVC
model = SVC(kernel='rbf', C=1, gamma='scale')
model.fit(X_train, y_train)
pred = model.predict(X_test)
scores = model.decision_function(X_test)   # für ROC
Im Kurs: Klassifikation in Kapitel 3 & Blatt 03 (auch Wälzlager, Digits).
SVR / LinearSVR sklearn.svm

Support-Vector-Regression — dieselbe Idee für kontinuierliche Zielwerte.

from sklearn.svm import SVR
model = SVR(kernel='rbf', C=10).fit(X, y)
Im Kurs: Regressions-Varianten der SVM.

5.2 Entscheidungsbäume

Interpretierbare Schwellwert-Regeln.

DecisionTreeClassifier / Regressor sklearn.tree

Entscheidungsbaum: teilt den Raum durch Schwellwert-Fragen. Interpretierbar, aber neigt zu Overfitting.

  • max_depth — maximale Tiefe (begrenzt Overfitting)
  • min_samples_leaf — Mindest-Samples pro Blatt
from sklearn.tree import DecisionTreeClassifier
tree = DecisionTreeClassifier(max_depth=4, random_state=42)
tree.fit(X, y)
Im Kurs: Basis-Lerner für Ensembles, Interpretierbarkeit (Kapitel 4/5).
plot_tree sklearn.tree

Zeichnet einen trainierten Entscheidungsbaum als Diagramm.

from sklearn.tree import plot_tree
plt.figure(figsize=(12, 6))
plot_tree(tree, filled=True, feature_names=namen)
plt.show()
Im Kurs: Baumstruktur visualisieren (Kapitel 4).

5.3 Ensemble-Methoden

Viele Modelle kombinieren.

RandomForestClassifier / Regressor sklearn.ensemble

Viele Entscheidungsbäume auf Bootstrap-Stichproben + zufälliger Feature-Auswahl. Mittelt → robust gegen Rauschen & Overfitting.

  • n_estimators — Anzahl Bäume
  • max_features — Features pro Split
  • feature_importances_ — gelernte Feature-Wichtigkeit
from sklearn.ensemble import RandomForestClassifier
rf = RandomForestClassifier(n_estimators=200, random_state=42)
rf.fit(X_train, y_train)
wichtig = rf.feature_importances_
Im Kurs: Robuste Klassifikation, Feature-Importance-Heatmaps (Blatt 05).
AdaBoostClassifier sklearn.ensemble

Boosting: trainiert Lerner sequenziell, jeder korrigiert die Fehler des vorigen.

from sklearn.ensemble import AdaBoostClassifier
ada = AdaBoostClassifier(n_estimators=50).fit(X, y)
Im Kurs: Boosting-Ensemble (Kapitel 5).

5.5 Die einheitliche API

Ein Muster für alle Modelle.

Die einheitliche sklearn-API Konzept

Jedes Modell folgt demselben Muster: .fit() trainiert, .predict() sagt vorher, .score() bewertet. Deshalb sind Modelle austauschbar.

model.fit(X_train, y_train)        # trainieren
y_pred = model.predict(X_test)     # Klasse vorhersagen
proba  = model.predict_proba(X_test)   # Wahrscheinlichkeiten
score  = model.decision_function(X_test)  # Konfidenz (SVM)
acc    = model.score(X_test, y_test)   # Accuracy
Im Kurs: Dieses Muster gilt für ALLE Modelle — einmal verstanden, überall anwendbar.
