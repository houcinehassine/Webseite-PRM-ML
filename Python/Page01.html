NumPy Grundlagen

numpy (als np) ist die Basis für alle numerischen Daten im Kurs. Feature-Matrizen, Labels, Zufallsdaten und das Raster für Entscheidungsflächen — alles sind NumPy-Arrays.

np.array linspace meshgrid boolesche Masken np.unique

Worum geht's?

Die wichtigsten NumPy-Bausteine, die in fast jeder Code-Zelle des Kurses vorkommen — mit Beispielen, wofür sie konkret eingesetzt werden.

1.1 Arrays erzeugen

Die Grundlage: Daten als ndarray.

np.array(liste) numpy

Erzeugt ein ndarray aus einer Python-Liste. Das zentrale Datenobjekt für Features (X) und Labels (y).

import numpy as np
X = np.array([[1, 2], [3, 4], [5, 6]])   # Form (3, 2)
y = np.array([0, 1, 0])
print(X.shape, y.shape)   # (3, 2) (3,)
Im Kurs: Grundbaustein jeder Feature-Matrix X (N×K) und jedes Label-Vektors y.
np.arange / np.linspace numpy

arange(start, stop, step) erzeugt gleichmäßige Werte mit Schrittweite, linspace(start, stop, n) erzeugt n Werte inklusive Endpunkt.

np.arange(0, 10, 2)        # [0 2 4 6 8]
np.linspace(0, 1, 5)       # [0. 0.25 0.5 0.75 1.]
ks = range(2, 11)          # für Elbow-Plots: k = 2..10
Im Kurs: linspace für Achsen/Mesh-Grids beim Plotten, arange für Indizes & Hyperparameter-Bereiche.
np.zeros / ones / full / *_like numpy

Erzeugt mit Nullen, Einsen oder einem festen Wert gefüllte Arrays. Die _like-Varianten übernehmen die Form eines vorhandenen Arrays.

np.zeros((2, 3))           # 2x3 Nullen
np.ones(5)                 # [1 1 1 1 1]
np.full(3, 7)              # [7 7 7]
np.zeros_like(X)           # Nullen in Form von X
Im Kurs: Initialisieren von Ergebnis-Arrays und Masken.
np.random.default_rng / normal / uniform numpy

Zufallszahlen. default_rng(seed) ist der moderne Generator; normal(loc, scale, size) erzeugt Gauß-Rauschen, uniform(low, high, size) Gleichverteilung.

rng = np.random.default_rng(42)
rauschen = rng.normal(0, 1, size=(100, 2))   # Gauß
ausreisser = rng.uniform(-3, 3, size=(10, 2))
Im Kurs: Künstliche Datensätze, Rauschen zu Features addieren (Blatt 05), reproduzierbar mit Seed.

1.2 Slicing, Indexing & Masken

Auf Teile von Arrays zugreifen und filtern.

Slicing & Indexing numpy

Zugriff auf Zeilen/Spalten. X[zeile, spalte], : wählt alles. Negative Indizes zählen von hinten.

X[0]          # erste Zeile
X[:, 0]       # erste Spalte (alle Zeilen)
X[:5]         # erste 5 Zeilen
X[-1]         # letzte Zeile
Im Kurs: Features/Labels aufteilen, Spalten auswählen.
Boolesche Masken numpy

Ein Vergleich liefert ein Array aus True/False, mit dem gefiltert wird — das Herzstück von Klassen-Scatter-Plots.

maske = (y == 1)
X[maske]              # nur Zeilen mit Label 1
X[y == k, 0]          # Feature 0 der Klasse k
ausreisser = labels == -1   # DBSCAN-Anomalien
Im Kurs: Klassen einfärben (X[y==k]), Anomalien filtern (labels == -1).
np.where(cond, a, b) numpy

Wählt elementweise zwischen zwei Werten je nach Bedingung — oder gibt mit einem Argument die Indizes der True-Stellen zurück.

np.where(imp < 0.01, 0, imp)   # kleine Werte auf 0 setzen
y_anom = np.where(labels == -1, 1, 0)
Im Kurs: Schwellwert-Masken (Blatt 05), Anomalie-Labels bauen.

1.3 Aggregationen & Statistik

Werte zusammenfassen — pro Achse oder gesamt.

np.unique(y, return_counts=True) numpy

Liefert die eindeutigen Werte und (optional) deren Häufigkeiten — die Standardmethode, um die Klassenverteilung zu prüfen.

klassen, anzahl = np.unique(y, return_counts=True)
for k, a in zip(klassen, anzahl):
    print(f'Klasse {k}: {a} Samples')
Im Kurs: Klassenbalance prüfen, stratify-Effekt zeigen (Blatt 02).
Aggregationen: mean, std, sum, median … numpy

Statistik über ganze Arrays oder entlang einer Achse (axis=0 = spaltenweise). Wichtig für Skalierung und Feature-Extraktion.

X.mean(axis=0)     # Spalten-Mittelwerte
X.std()            # Gesamt-Standardabweichung
np.median(werte)
np.sqrt((sig**2).mean())   # RMS eines Signals
Im Kurs: Feature-Extraktion (Mittelwert, Std, RMS), Skalierung, Rausch-Stärke noise·std(X).
argmax / argmin / argsort / bincount numpy

argmax/argmin geben den Index des Maximums/Minimums; argsort die Sortier-Indizes; bincount zählt Ganzzahl-Häufigkeiten.

best_k = ks[np.argmin(bics)]        # bestes Modell via BIC
top3 = np.argsort(importances)[::-1][:3]
np.bincount(labels)                 # Clustergrößen
Im Kurs: Bestes k/Hyperparameter wählen, Top-Features, Clustergrößen.

1.4 Kombinieren & Mesh-Grid

Arrays zusammenfügen und das Gitter für Plots bauen.

np.meshgrid + np.c_ numpy

meshgrid erzeugt ein Koordinatengitter; np.c_ stapelt Spalten. Zusammen bilden sie das Raster für Entscheidungsflächen.

xx, yy = np.meshgrid(np.linspace(x0, x1, 300),
                     np.linspace(y0, y1, 300))
gitter = np.c_[xx.ravel(), yy.ravel()]
Z = model.predict(gitter).reshape(xx.shape)
Im Kurs: Entscheidungsgrenzen visualisieren (SVM, Blatt 03 / Kapitel 3).
np.vstack / np.concatenate numpy

Arrays zusammenfügen — vstack stapelt zeilenweise, concatenate entlang einer beliebigen Achse.

X = np.vstack([X_normal, X_anomalien])
alle = np.concatenate([a, b])
Im Kurs: Normale + anomale Messpunkte kombinieren (Blatt 04).
np.dot / np.linalg / np.interp numpy

dot = Skalar-/Matrixprodukt, linalg.norm = Vektorlänge, interp = lineare Interpolation.

np.dot(w, x)               # Skalarprodukt
np.linalg.norm(a - b)      # euklidischer Abstand
tpr_i = np.interp(mean_fpr, fpr, tpr)   # ROC mitteln
Im Kurs: Abstände, ROC-Kurven über Folds mitteln (Kapitel 7).
np.save / np.load numpy

Speichert/lädt Arrays im .npy-Format (binär, schnell).

np.save('daten.npy', X)
X = np.load('daten.npy')
Im Kurs: Vorberechnete Datensätze laden (z.B. anomaly_detection.npy).
