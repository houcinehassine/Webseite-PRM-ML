3.1 Synthetische Datensätze

Künstliche Daten mit bekannten Eigenschaften — perfekt zum Üben.

make_blobs sklearn.datasets

Erzeugt gut trennbare, kugelförmige Cluster (Gauß-Blobs). Der meistgenutzte Spielzeug-Datensatz im Kurs.

  • n_samples — Anzahl Punkte
  • centers — Anzahl Cluster/Zentren
  • cluster_std — Streuung pro Cluster
  • random_state — Reproduzierbarkeit
from sklearn.datasets import make_blobs
X, y = make_blobs(n_samples=300, centers=4,
                  cluster_std=0.5, random_state=42)
Im Kurs: Clustering (KMeans, GMM, DBSCAN), Sensordaten simulieren.
make_classification sklearn.datasets

Erzeugt einen Klassifikationsdatensatz mit steuerbarer Feature-Anzahl, Klassen und Informationsgehalt.

  • n_informative — aussagekräftige Features
  • n_redundant — redundante Features
  • n_clusters_per_class — Cluster je Klasse
from sklearn.datasets import make_classification
X, y = make_classification(n_samples=200, n_features=2,
        n_classes=3, n_clusters_per_class=1, n_redundant=0,
        random_state=42)
Im Kurs: Klassifikator-Übungen, saubere 2D-Daten zum Plotten (Blatt 02).
make_moons / make_circles sklearn.datasets

Nicht-linear trennbare Formen (Halbmonde, verschachtelte Kreise) — ideal, um RBF-Kernel und dichtebasiertes Clustering zu testen.

from sklearn.datasets import make_moons, make_circles
X, y = make_moons(n_samples=500, noise=0.08, random_state=0)
Im Kurs: DBSCAN vs. GMM (Blatt 06), nichtlineare SVM-Grenzen.
make_regression sklearn.datasets

Erzeugt ein lineares Regressionsproblem mit Rauschen.

from sklearn.datasets import make_regression
X, y = make_regression(n_samples=200, n_features=5,
                       random_state=42)
Im Kurs: Skalierung & Data-Leakage demonstrieren (Blatt 02).

3.2 Echte Datensätze

Eingebaute Standard-Datensätze.

load_digits sklearn.datasets

8×8-Bilder handgeschriebener Ziffern (0–9), als 64-Feature-Vektoren. Klein genug, um offline im Browser zu laufen.

from sklearn.datasets import load_digits
X, y = load_digits(return_X_y=True)   # X: (1797, 64)
digits = load_digits()
bild = digits.images[0]               # 8x8
Im Kurs: SVM-/RandomForest-Klassifikation, PCA+GMM-Bildgenerator, Feature-Importance-Heatmap.
load_iris sklearn.datasets

Der klassische Iris-Datensatz: 150 Blüten, 4 Features, 3 Arten.

from sklearn.datasets import load_iris
X, y = load_iris(return_X_y=True)
Im Kurs: Einfache Klassifikations-Demos.
fetch_lfw_people / fetch_olivetti_faces sklearn.datasets

Lädt echte Gesichts-Datensätze (Download nötig). Hochdimensional → meist mit PCA reduziert.

from sklearn.datasets import fetch_lfw_people
faces = fetch_lfw_people(min_faces_per_person=70)
X = faces.data
Im Kurs: Gesichtserkennung (SVM) & Gesichts-Generierung (PCA+GMM). Hinweis: braucht Internet.

3.3 pandas

Tabellen laden und inspizieren.

pd.DataFrame / pd.Series pandas

Tabellarische Datenstruktur mit benannten Spalten (DataFrame) bzw. eine einzelne Spalte (Series).

import pandas as pd
df = pd.DataFrame(X, columns=['f1', 'f2', 'f3'])
df['label'] = y
print(df.head())
print(df.describe())
Im Kurs: Daten inspizieren, Feature-Tabellen aufbauen.
pd.read_csv / df.quantile pandas

read_csv lädt eine CSV-Datei; quantile berechnet Perzentile (z.B. für Ausreißergrenzen).

df = pd.read_csv('songs.csv')
grenze = df['tempo'].quantile(0.95)
Im Kurs: Reale Datensätze laden (Songs, Citizen-DB), Schwellen bestimmen.

3.4 seaborn

Statistische Visualisierung in einer Zeile.

sns.heatmap / clustermap seaborn

Visualisiert eine Matrix farbcodiert. clustermap ordnet zusätzlich ähnliche Zeilen/Spalten gruppiert an.

import seaborn as sns
sns.heatmap(confusion_matrix(y, pred), annot=True, fmt='d')
sns.heatmap(param_grid_scores)   # Hyperparameter-Suche
Im Kurs: Confusion Matrix, Hyperparameter-Heatmap für DBSCAN-Suche (Blatt 04).
sns.pairplot / sns.set seaborn

pairplot zeichnet alle Feature-Paare gegeneinander (Streudiagramm-Matrix); sns.set() setzt ein schöneres Standard-Theme.

sns.set()
sns.pairplot(df, hue='label')   # nach Klasse einfärben
Im Kurs: Feature-Verteilungen & Cluster überblicken (Blatt 04 Microtargeting).
