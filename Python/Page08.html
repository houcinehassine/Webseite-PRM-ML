8.1 Klassifikations-Metriken

Accuracy, F1 und die Confusion Matrix.

accuracy_score / f1_score sklearn.metrics

accuracy = Anteil korrekt; f1 = harmonisches Mittel aus Precision & Recall (besser bei Unbalance).

from sklearn.metrics import accuracy_score, f1_score
acc = accuracy_score(y_test, pred)
f1  = f1_score(y_test, pred, average='macro')
Im Kurs: Standard-Bewertung von Klassifikatoren.
confusion_matrix sklearn.metrics

Kreuztabelle aus wahren vs. vorhergesagten Klassen — die Basis aller Klassifikationsmetriken (TP/FP/TN/FN).

from sklearn.metrics import confusion_matrix
cm = confusion_matrix(y_test, pred)
sns.heatmap(cm, annot=True, fmt='d')
Im Kurs: Fehleranalyse, Grundlage für TPR/FPR (Kapitel 7).

8.2 ROC, AUC & Regression

Schwellwertunabhängige Bewertung & MSE.

roc_curve / auc / roc_auc_score sklearn.metrics

ROC-Kurve trägt TPR gegen FPR über alle Schwellwerte auf; AUC ist die Fläche darunter (1.0 = perfekt, 0.5 = Zufall).

from sklearn.metrics import roc_curve, auc, roc_auc_score
fpr, tpr, thr = roc_curve(y_test, scores)
flaeche = auc(fpr, tpr)
# kurz:
roc_auc_score(y_test, scores)
Im Kurs: Klassifikator-Güte schwellwertunabhängig bewerten (Kapitel 7).
RocCurveDisplay / precision_recall_curve sklearn.metrics

Fertige ROC-Darstellung bzw. die Precision-Recall-Kurve (besser bei starker Klassen-Unbalance).

from sklearn.metrics import RocCurveDisplay, precision_recall_curve
RocCurveDisplay.from_estimator(model, X_test, y_test)
prec, rec, thr = precision_recall_curve(y_test, scores)
Im Kurs: ROC schnell plotten, PR-Kurve bei seltenen positiven Fällen.
mean_squared_error sklearn.metrics

Mittlerer quadratischer Fehler — die Standardmetrik für Regression.

from sklearn.metrics import mean_squared_error
mse = mean_squared_error(y_test, pred)
Im Kurs: Regressionsmodelle bewerten.

8.3 Clustering-Metriken

Cluster-Güte ohne Labels messen.

silhouette_score / silhouette_samples sklearn.metrics

Misst, wie gut Cluster getrennt sind (−1…+1, größer = besser). samples liefert den Wert pro Punkt.

from sklearn.metrics import silhouette_score
s = silhouette_score(X, labels)
Im Kurs: Clustering-Qualität bewerten (Blatt 04).
davies_bouldin_score sklearn.metrics

Cluster-Gütemaß — hier gilt: kleiner ist besser (0 = ideal).

from sklearn.metrics import davies_bouldin_score
db = davies_bouldin_score(X, labels)
Im Kurs: Clusterzahl vergleichen (Blatt 04).

8.4 Feature-Importance

Welche Features zählen?

feature_importances_ sklearn.ensemble

Attribut trainierter Baum-Ensembles: wie stark jedes Feature die Unreinheit reduziert (über alle Bäume gemittelt).

rf.fit(X, y)
wichtig = rf.feature_importances_
top3 = np.argsort(wichtig)[::-1][:3]
plt.imshow(wichtig.reshape(8, 8), cmap='hot')
Im Kurs: Welche Pixel/Features nutzt das Modell? Heatmap (Blatt 05).
permutation_importance sklearn.inspection

Modell-unabhängig: misst den Score-Abfall, wenn ein Feature zufällig vertauscht wird.

from sklearn.inspection import permutation_importance
r = permutation_importance(model, X_test, y_test, n_repeats=10)
print(r.importances_mean)
Im Kurs: Feature-Wichtigkeit für beliebige Modelle (Kapitel 5, Blatt 05).
