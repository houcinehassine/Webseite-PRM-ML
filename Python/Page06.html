6.1 KMeans & DBSCAN

Partitionierendes vs. dichtebasiertes Clustering.

KMeans sklearn.cluster

Teilt Daten in k kugelförmige Cluster, indem die Distanz zu den Clusterzentren minimiert wird (Inertia).

  • n_clusters — Anzahl Cluster k
  • n_init — Anzahl Neustarts
  • inertia_ — Summe quadrierter Distanzen (Elbow)
  • labels_ — Cluster-Zuordnung
from sklearn.cluster import KMeans
km = KMeans(n_clusters=4, n_init=10, random_state=42)
km.fit(X)
labels = km.labels_
zentren = km.cluster_centers_
inertia = km.inertia_   # für Elbow-Plot
Im Kurs: Segmentierung & Microtargeting (Kapitel 4, Blatt 04).
DBSCAN sklearn.cluster

Dichtebasiertes Clustering: findet beliebig geformte Cluster und markiert Ausreißer mit Label -1. Braucht KEINE Clusterzahl.

  • eps — Radius der Nachbarschaft
  • min_samples — Mindest-Nachbarn für Core Point
  • labels_ == -1 — Rauschen / Anomalien
from sklearn.cluster import DBSCAN
db = DBSCAN(eps=0.9, min_samples=5).fit(X)
anomalien = (db.labels_ == -1)
Im Kurs: Anomalie-Erkennung (Blatt 04), Halbmonde (Blatt 06).
SpectralClustering / kneighbors_graph sklearn.cluster

Graph-/ähnlichkeitsbasiertes Clustering für nicht-konvexe Strukturen.

from sklearn.cluster import SpectralClustering
sc = SpectralClustering(n_clusters=2).fit_predict(X)
Im Kurs: Alternative zu KMeans bei komplexen Formen (Kapitel 4).

6.2 Gaussian Mixture Models

Weiche, generative Cluster.

GaussianMixture sklearn.mixture

Modelliert Daten als Mischung von Gauß-Verteilungen (Ellipsen). Liefert weiche Zuordnungen und kann neue Daten generieren.

  • n_components — Anzahl Gauß-Komponenten
  • covariance_type — 'full'/'tied'/'diag'/'spherical'
  • bic(X) / aic(X) — Kriterium zur Wahl von n_components
  • sample(n) — n neue Datenpunkte generieren
from sklearn.mixture import GaussianMixture
gmm = GaussianMixture(n_components=3,
        covariance_type='full', random_state=42)
gmm.fit(X)
labels = gmm.predict(X)
proba  = gmm.predict_proba(X).max(axis=1)
neue, _ = gmm.sample(12)        # generieren
print(gmm.bic(X), gmm.aic(X))   # Modellwahl
Im Kurs: Betriebszustände, Bildgenerator, MRT-Segmentierung (Kapitel 6, Blatt 06).
Modellwahl per BIC/AIC Rezept

Die optimale Komponenten-/Clusterzahl wählt man, indem man BIC über mehrere k berechnet und das Minimum nimmt.

bics = [GaussianMixture(k, random_state=42).fit(X).bic(X)
        for k in range(1, 11)]
best_k = np.argmin(bics) + 1
